# Site Pessoal

Projeto estático (HTML/CSS/JS) com Tailwind CSS para geração do arquivo `output.css`.

## Como abrir o projeto

#### Pré-requisitos

- Node.js 18+ instalado.

#### Passos

1. Instale as dependências:

```bash
npm install
```

2. Inicie o Tailwind em modo watch:

```bash
npm run dev
```

3. Com o comando em execução, abra o arquivo `src/index.html` no navegador.

Sempre que você alterar `src/assets/css/main.css`, o arquivo `src/output.css` será atualizado automaticamente.
