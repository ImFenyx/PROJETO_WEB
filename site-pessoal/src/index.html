Meu principal objetivo é contribuir ativamente em projetos impactantes e inovadores. Possuo foco em produtividade, trabalho em equipe, otimização e segurança. Comecei na programação em 2023, e desde então, tenho me dedicado a aprender e aprimorar minhas habilidades. Projetos

Concrete Jungle

Jogo de plataforma 2D desenvolvido em GameMaker 2, feito na Etec São José dos Campos. O projeto foi criado como parte de um curso de desenvolvimento de jogos e é um exemplo do meu interesse em design de jogos e desenvolvimento de software.

F-Tools Project

Um projeto Python no terminal para múltiplas ferramentas de diversas categorias, incluindo automação, programação, cálculos, cores, unidades, texto, randomização, entre outras. O projeto é uma coleção de utilitários que visam facilitar tarefas comuns e melhorar a produtividade em diferentes áreas.

EGProj

(Colaborador) Um sistema em Python por terminal para praticar e aprimorar o vocabulário de inglês de forma interativa e autônoma. Este projeto foi desenvolvido de forma colaborativa para servir como uma ferramenta prática de aprendizado, enquanto capacita os participantes em programação Python, boas práticas de desenvolvimento e metodologias ágeis.

Fenyx's Core Utils

Um projeto Python de software livre que visa criar uma coleção de utilitários essenciais para sistemas operacionais baseados em Linux. O projeto é inspirado nos GNU Core Utilities e tem como objetivo fornecer ferramentas de linha de comando eficientes, confiáveis e fáceis de usar para usuários e desenvolvedores.
